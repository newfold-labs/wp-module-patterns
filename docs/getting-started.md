# Getting started

Prerequisites: PHP 7.3+, Composer. The module requires wp-module-data, wp-module-features, wp-module-installer.

```bash
composer install
composer run test
composer run lint
composer run clean
```

See [integration.md](integration.md) for using in a host plugin.
