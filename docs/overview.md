# Overview

**wp-module-patterns** provides WordPress Cloud Patterns for Newfold brand plugins. It registers with the Newfold Module Loader and depends on wp-module-data, wp-module-features, and wp-module-installer. Used by onboarding and other modules for pattern content. Maintained by Newfold Labs. Distributed via Newfold Satis.
